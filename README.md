# Trimly - Weight Loss Calculator

A progressive web app for tracking weight loss progress and calculating personalized weight loss plans.

## Features

- Personalized weight loss plan calculation
- Daily check-ins and progress tracking
- Streak tracking and milestones
- PWA for offline use
- Responsive design

## Getting Started

Open `trimly-v2.html` in a modern web browser to use the app.

For development, you can serve the file locally using a simple HTTP server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx serve .

# Then open http://localhost:8000/trimly-v2.html
```

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage for data persistence
- PostHog for analytics (configure PH_KEY)

## Development

The app is currently a single HTML file with inline CSS and JavaScript. For larger development, consider separating into multiple files.
